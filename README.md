# Personal Web Profile - CMSC 207 Mini-Project

## Project Overview
This is a personal web profile created for the CMSC 207 Mini-Project. It demonstrates semantic HTML structure, CSS styling with a modern color palette, and basic JavaScript interactivity.

## Features
* **Semantic HTML5:** Uses header, nav, section, and footer tags.
* **CSS Styling:** Custom gradients, hover effects, and a responsive grid layout.
* **JavaScript:** Includes a "Fun Fact" generator button.
* **Mobile Responsiveness:** Adapts layout for smaller screens.

## Technologies Used
* HTML5
* CSS3
* JavaScript
